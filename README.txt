Adds the vivenio event registration to your wordpress page.

== Description ==

This plugin adds your vivenio event registration to your wordpress page. You can enter your event URL in the settings and insert the event registration with an inser tag into your WordPress page.

== Installation ==

1. Upload `vivenio-integration.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Define your event base url in "settings => vivenio" for example: https://your-event-domain.de/event
4. Place `[vivenio-event event="demo"]` in your page. And change "demo" to your event key.

= System requirements =
* PHP 7.1 (Recommended 7.2)
